# Gmail ChatGPT Reply Assistant v0.5.0

Gmailのメールスレッドを取得してChatGPT用プロンプトを作り、ChatGPTで作成した返信案をGmailの返信欄または新規作成欄へ挿入するChrome拡張機能です。

This Chrome extension extracts a Gmail thread, creates a prompt for ChatGPT, and inserts the reply generated by ChatGPT back into a Gmail reply box or compose window.

## v0.5.0 の変更点 / Changes in v0.5.0

- 別のGmailスレッドへ移動した際、前のスレッド本文と作成済みプロンプトを自動的に破棄します。
- When Gmail navigates to another thread, the previous extracted thread and generated prompt are discarded automatically.
- 「プロンプトをコピーしてChatGPTを開く」と「プロンプトだけコピー」は、表示中スレッドからプロンプトを再作成します。抽出スレッド欄を手動編集している場合は、その編集内容を維持します。
- The copy actions rebuild the prompt from the currently displayed thread. Manual edits in the extracted-thread field are preserved for the same thread.
- クリップボードへ書き込んだ後、実際のクリップボード内容が今回作成したプロンプトと一致するかを読み戻して確認します。一致しなければ再試行し、確認できない場合はChatGPTを開きません。
- After writing, the extension reads the clipboard back and verifies that it exactly matches the newly generated prompt. It retries on mismatch and does not open ChatGPT if verification fails.
- プロンプト処理中は操作ボタンを一時的に無効化し、連打や並行処理による古い内容の上書きを防止します。
- Action buttons are temporarily disabled during prompt processing to prevent overlapping operations from restoring stale content.
- コピー確認後のChatGPT起動はManifest V3のサービスワーカーから新しいタブを作成する方式に変更しました。
- ChatGPT is now opened from a Manifest V3 service worker after clipboard verification.

## v0.4.0 から継続する機能 / Features retained from v0.4.0

- 日本語環境では日本語UI、それ以外の言語環境では英語UIに切り替わります。
- The UI is Japanese in Japanese browser environments and English in all other language environments.
- Gmail上のパネル、ステータスメッセージ、生成プロンプトも日本語/英語で切り替わります。
- 「メールの出力言語（選択）」と「メールの出力言語（自由入力・任意）」を利用できます。
- The email output language selector and optional custom language input are available.
- 自由入力欄に言語指定を書いた場合は、選択欄より自由入力欄が優先されます。
- Custom language instructions override the selected language.

## インストール / Installation

1. ZIPを解凍します。 / Unzip the package.
2. Chromeで `chrome://extensions/` を開きます。 / Open `chrome://extensions/` in Chrome.
3. 「デベロッパーモード」をオンにします。 / Turn on Developer mode.
4. 旧版を読み込んでいる場合は削除するか、解凍先をv0.5.0へ入れ替えます。 / Remove the old unpacked version or replace its folder with v0.5.0.
5. 「パッケージ化されていない拡張機能を読み込む」を押します。 / Click “Load unpacked”.
6. 解凍した `gmail-chatgpt-assistant` フォルダを選択します。 / Select the unzipped `gmail-chatgpt-assistant` folder.
7. Gmailを再読み込みします。 / Reload Gmail.

## 使い方 / Usage

1. Gmailで返信したいメールスレッドを開きます。 / Open the Gmail thread you want to reply to.
2. 右下の **GPT Mail** ボタンを押します。 / Click the **GPT Mail** button at the bottom right.
3. 必要に応じて用途、文体、メールの出力言語、追加条件を指定します。 / Choose the use case, tone, email output language, and any additional instructions.
4. 「プロンプトをコピーしてChatGPTを開く」を押します。表示中の最新スレッドを取得し、コピー内容を照合してからChatGPTを開きます。 / Click “Copy prompt and open ChatGPT.” The extension loads the current thread, verifies the copied prompt, and then opens ChatGPT.
5. ChatGPTに貼り付けて返信案を作成します。 / Paste the prompt into ChatGPT and generate the reply.
6. ChatGPTの最終返信案だけをコピーします。 / Copy only the final reply from ChatGPT.
7. Gmailへ戻り、「クリップボードから読み込む」を押します。 / Return to Gmail and click “Read from clipboard”.
8. 内容を確認して「Gmailへ挿入して下書き化」を押します。 / Review it, then click “Insert into Gmail draft”.

## メールの出力言語 / Email output language

選択欄では次を選べます。

- 元メールに合わせる / Match source email
- 日本語 / Japanese
- 英語 / English
- 日英併記 / Japanese + English

自由入力欄には、より細かい指定を書けます。

Examples:

- フランス語
- 簡体字中国語
- 相手の言語に合わせつつ、件名だけ英語
- Spanish, but keep names and product terms in English

自由入力欄に何か書かれている場合、プロンプトでは自由入力欄の内容が優先されます。

## 制限 / Limitations

- この拡張機能はメール本文を外部サーバーへ自動送信しません。ユーザーがChatGPT画面へ貼り付けた時点でChatGPT側に送信されます。
- This extension does not automatically send email content to an external server. Email content is sent to ChatGPT only when you paste the prompt into ChatGPT.
- クリップボードを読み戻せない環境では、安全のためChatGPTを自動的に開きません。画面内のプロンプト欄から手動でコピーしてください。
- If clipboard read-back is unavailable, ChatGPT is not opened automatically for safety. Copy the prompt manually from the prompt field.
- GmailのDOM構造に依存するため、Gmail側のUI変更でスレッド抽出や返信欄挿入が失敗する可能性があります。
- The extension depends on Gmail’s DOM structure, so Gmail UI changes may break thread extraction or draft insertion.
- 送信は自動では行いません。必ず内容を確認してから手動で送信してください。
- It never sends emails automatically. Always review and send manually.
